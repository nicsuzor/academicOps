# /session-analyzer

Analyze Claude Code sessions and produce a curated daily summary.

## Arguments

- `$ARGUMENTS` - Optional: date (YYYYMMDD) or "today" (default)

## Invocation

```
Skill(skill="session-analyzer")
```

Pass `$ARGUMENTS` to the skill for date filtering.
