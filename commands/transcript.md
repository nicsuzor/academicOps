# Transcript Generator

Generate markdown transcripts from Claude Code session files.

## Usage

```
Bash("uv run python $AOPS/scripts/claude_transcript.py <session.jsonl> -o <output-base>")
```

## Find session files

```
Bash("ls -lt ~/.claude/projects/-$(basename $(pwd))/*.jsonl | grep -v agent | head -1")
```

## Output

- Location: `$ACA_DATA/sessions/claude/`
- Naming: `YYYYMMDD-<project>-<slug>-{full,abridged}.md`
- Two versions generated by default (use `--full-only` or `--abridged-only` for one)
