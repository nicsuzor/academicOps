# Commands

## Overview

15 slash commands providing strategic workflows, task management, and framework operations. Commands are invoked with `/command-name` syntax.

## Strategic & Planning

- [[meta.md|/meta]] - Strategic supervisor for academicOps framework - loads complete context, makes principled decisions, handles work end-to-end with TDD quality gates
- [[supervise.md|/supervise]] - Generic supervisor loading workflow templates for orchestrated multi-agent tasks (tdd, batch-review, skill-audit)
- [[academicOps/commands/strategy|/strategy]] - Strategic thinking partner for exploration, planning, and clarity - facilitates thinking without executing tasks
- [[qa.md|/qa]] - Quality assurance agent that verifies concrete outcomes against user goals and acceptance criteria using real-world data
- [[ttd.md|/ttd]] - Alias for `/supervise tdd` - test-first development workflow

## Learning & Improvement

- [[log.md|/log]] - Log agent performance (successes and failures) to framework learning system
- [[learn.md|/learn]] - Make minimal, graduated framework tweaks with experiment tracking
- [[consolidate.md|/consolidate]] - Consolidate LOG.md entries into thematic learning files and archive

## Task Management

- [[add.md|/add]] - Quick-add a task or checklist item from session context
- [[pull.md|/pull]] - Get shit done! Process high-priority tasks sequentially, finding relevant context via memory server
- [[email.md|/email]] - Extract action items from emails and create tasks automatically with context-aware categorization

## Batch Processing & Automation

- [[parallel-batch.md|/parallel-batch]] - Orchestrate parallel batch processing of files with automatic skill delegation, question batching, and progress tracking
- [[review-training-cmd.md|/review-training-cmd]] - Process one matched review/source pair to extract training data

## Visualization & Diagnostics

- [[task-viz.md|/task-viz]] - Generate visual mind-map of tasks, projects, and goals using automated layout via excalidraw
- [[diag.md|/diag]] - Quick diagnostic check of what's currently loaded in this session
