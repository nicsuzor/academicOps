# Project Independence

**Statement**: Projects must work independently without cross-dependencies.

## Derivation

Coupling projects creates fragile systems where changes cascade unpredictably. Each project should be self-contained.

Evidence of violations and corrections tracked in bd issues (label: `learning`).
