# Data Boundaries

**Statement**: NEVER expose private data in public places. Everything in this repository is PRIVATE unless explicitly marked otherwise.

## Derivation

Privacy is a fundamental right. Accidental exposure of private data causes irreversible harm.

## Evidence

Evidence of violations and corrections tracked in bd issues (label: `learning`).

## Enforcement

Hard gate via `settings.json` deny rules at PreToolUse. See [[enforcement-map.md]].
