## Enhanced Stitch Prompt

Overwhelm Dashboard for an academic researcher with ADHD. The app manages complex task hierarchies and agent session workflows in a single pane of glass. Currently, the dashboard uses an 'Operator Theme' with amber (#f2aa0d) highlights, dark grey (#0A0A0A) backgrounds, a 12-column Bento Grid layout, and JetBrains Mono / Space Grotesk typography.

We need to improve the UX and information density for the 'Threaded Tasks' view and the 'Task Graph' view.

**Step 1:** Refine the layout of the 'Threaded Tasks' split view. Redesign the left-hand 'Directory Tree' to be less visually overwhelming and more scannable, using subtle indentations and muted colors for collapsed folders.
**Step 2:** On the 'Task Graph' screen, add a new suite of interactive, highly-visible filtering controls (status checkboxes, priority toggles, project tags) to the top navigation bar without breaking the 12-column grid structure.
**Step 3:** Ensure all new components adhere strictly to the dark amber Operator aesthetic, using thin (1px) amber borders, minimal padding, and subtle glassmorphic panels to maintain a technical, high-density "terminal" vibe.
