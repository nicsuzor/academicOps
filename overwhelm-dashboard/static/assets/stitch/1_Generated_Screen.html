Grid-based terminal aesthetic. Low-fidelity, high-functional density. Designed for rapid data entry and command-line interactions.

Layout Principles: The Grid

Every element must align to a strict 8px/24px grid. No shadows. Depth is communicated through border thickness and nesting levels. All typography is monospaced.
