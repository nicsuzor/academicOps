[AOPS: Session Reflection]

Before ending, you MUST ensure that you:

1. Commit and push any pending changes to Github

2. Run session reflection to capture learnings:

```
Skill(skill='session-insights', args='current')
```

The user is expecting these in order and will skip by interrupting with Ctrl+C if they choose to exit quickly.
