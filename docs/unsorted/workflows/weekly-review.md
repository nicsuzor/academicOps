# Weekly Review Workflow

## Purpose

Process accumulated ideas, assess progress, and plan the upcoming week.

## When

Friday afternoon or Sunday (consistent time each week)

## Workflow Steps

### 1. PROCESS INBOX (10 mins)

```
1. Open Microsoft To-Do "Random Ideas" list
2. For each item:
   - Categorize and assess (Impact/Effort)
   - Link to project if applicable
3. Clear To-Do inbox
```

### 2. PROJECT REVIEW (15 mins)

```
For each Active Project:
1. Update progress percentage
2. Review completed tasks
3. Identify blockers
4. Set next week's priority level
5. Update "Next Action" field
```

### 3. WEEK REFLECTION (5 mins)

Create Progress Tracking entry:

```
- Goals Set: [What you planned]
- Goals Achieved: [What got done]
- Key Accomplishments: [Wins to celebrate]
- Challenges: [What blocked progress]
- Energy Level: [High/Medium/Low]
- Projects Advanced: [Link projects worked on]
```

### 4. NEXT WEEK PLANNING (10 mins)

```
1. Identify top 3 project priorities
2. Set weekly focus areas
3. Block time for Deep Work
4. Schedule any project milestones
5. Pre-select Monday's tasks
```

### 5. SYSTEM MAINTENANCE (5 mins)

```
- Archive completed projects
- Update project deadlines
- Clear completed tasks older than 2 weeks
- Check for orphaned tasks (no project)
```

## Review Templates

### Quick Stats

- Tasks completed this week: X
- Projects advanced: Y
- Ideas captured: Z

### Focus Questions

- What's the ONE thing that must happen next week?
- Which project needs the most attention?
- What quick wins can build momentum?

## Success Metrics

- Complete review in < 45 minutes
- All ideas processed from inbox
- Clear priorities for next week
- No tasks/ideas lost in system

## If Behind Schedule

- Shortened version (15 mins):
  1. Process inbox only
  2. Set top 3 tasks for Monday
  3. Schedule full review for next week
